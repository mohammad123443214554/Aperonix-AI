📋 Setup Guide
  1. Create a GitHub repo and upload your brain.json file to it.
  2. Generate a GitHub Personal Access Token with repo scope.
  3. Fill in the GitHub Configuration above and click Save.
  4. Go to Teach AI, select a Teacher, paste its API key, and enter a topic.
  5. The Teacher AI will generate code and Aperonix will learn it automatically!
  6. Deploy this project to Vercel — the /api/brain.js function handles everything.
