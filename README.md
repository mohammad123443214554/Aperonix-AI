# 🤖 Aperonix-AI

> World's first AI built from scratch using GitHub data — by a 14-year-old developer from India!

## What is Aperonix?
Aperonix is an AI that learns from GitHub — no API keys, no paid tools, built from zero!

## How it works
- Blank brain (like a newborn baby)
- Learns from GitHub data (1B+ examples)
- Has ethics filter (won't help with bad things)
- Built with pure JavaScript — no libraries!
